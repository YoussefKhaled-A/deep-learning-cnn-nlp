# 🧠 Deep Learning Coursework — CNN Image Classification, Audio & NLP

A collection of deep-learning notebooks built for an **Advanced Human-Computer Interaction** course
(25CSCI20H, British University in Egypt), covering computer vision, audio, and natural-language tasks.

> 🎓 Academic coursework — each notebook is a self-contained lab with code, results, and notes.

## What's inside
- **CNN fundamentals** — building and training a convolutional neural network from scratch.
  *(`Advanced_HCI_Python_CNN_Tutorial_1.ipynb`)*
- **Transfer learning with VGG19** — fine-tuning a pre-trained ImageNet model for image
  classification. *(`Advanced_HCI_Python_CNN_VGG19_Lab4.ipynb`)*
- **Audio classification (Chroma / CQT)** — extracting audio features with `librosa` and feeding a CNN.
  *(`Advanced_HCI_Python_CNN_ChromaCQT_Lab3.ipynb`)*
- **NLP with BERT** — text processing and classification using transformer models.
  *(`Advanced_HCI_Python_NLP.ipynb`)*
- Supporting lab reports (PDF) and slides.

## Tech
`Python` · `TensorFlow` / `Keras` · `PyTorch` · `librosa` · `Hugging Face Transformers (BERT)` · `scikit-learn`

## Run
```bash
pip install tensorflow torch transformers librosa scikit-learn jupyter
jupyter notebook
```
> Note: training benefits from a GPU; the VGG19 and BERT notebooks are heaviest.
